Ingeniería y Limpieza de Datos

El 80% del trabajo de un Científico de Datos. Análisis, diagnóstico y saneamiento exhaustivo del dataset Salud.csv evaluando múltiples estrategias de corrección.

1. Diagnóstico de Anomalías

Los algoritmos matemáticos no tienen sentido común. Si reciben la palabra "180mg" cuando esperan un número, el programa colapsará. Al revisar nuestro archivo crudo, detectamos ruido sistemático que requiere limpieza estructural.

Errores Críticos Detectados

  • Tipos Mixtos: 180mg en Glucosa, 45 años en Edad.
  • Errores de Localización: Uso de coma europea "30,5" en lugar de punto.
  • Ruido Categórico: " baja ", "NO", "si".
  • Valores Nulos (NaN): Celdas vacías que rompen las matemáticas del modelo.

2. Catálogo de Estrategias de Limpieza

En Machine Learning no hay una sola forma de limpiar datos. Dependiendo del tamaño de nuestro dataset y del problema, debemos elegir la técnica correcta.

Imputación (fillna)

Reemplaza los valores nulos calculando la media, mediana (números) o moda (textos) del resto de la columna.

Ideal para: Datasets pequeños donde no podemos darnos el lujo de perder información de otras columnas útiles.

Riesgo: Puede crear "datos falsos" y reducir la varianza natural si se abusa de ella.

Eliminación (dropna)

Borra la fila (paciente) entera si le falta tan solo un dato, o si el dato es insalvable.

Ideal para: Datasets enormes (millones de registros) donde borrar 50 filas no afecta estadísticamente.

Riesgo: Pérdida masiva de datos. Si borras a un paciente porque no tiene "Edad", también pierdes sus datos valiosos de "Glucosa" y "Presión".

Saneamiento Estructural

Uso de Expresiones Regulares (Regex) para extraer números del texto y eliminación de filas repetidas accidentalmente.

Obligatorio: Transforma el texto sucio ("180mg") en variables numéricas operables (180.0) y elimina clones (drop_duplicates).

3. Implementación del Pipeline (Código Python)

En este código aplicamos todas las reglas: Primero quitamos a los pacientes que estén repetidos en la lista. Luego, le quitamos las letras a los números (borramos "mg" y "años"). Finalmente, le damos a elegir al código: si a un paciente le falta un dato, le ponemos el promedio; pero dejamos el código listo por si el jefe nos pide mejor borrar a esos pacientes.

Explicación del Flujo:

  • drop_duplicates(): Garantiza que no haya sobre-representación de muestras idénticas.
  • str.replace() y astype(float): Transforma las Series de tipo 'Object' a 'Float64'.
  • Mostramos ambas estrategias de nulos: Imputación (Mediana para numéricos, Moda para categóricos) y Eliminación Listwise (dropna), dejando la última comentada como alternativa arquitectónica. 

import pandas as pd
import numpy as np

# 1. Cargar el dataset crudo
df = pd.read_csv('Documentos/Salud.csv')

# ==========================================
# FASE 1: DUPLICADOS
# ==========================================
# Elimina filas que sean exactamente iguales (clones accidentales)
df.drop_duplicates(inplace=True)

# ==========================================
# FASE 2: LIMPIEZA ESTRUCTURAL (STRING A FLOAT)
# ==========================================
# Cambiar coma por punto en el IMC ("30,5" -> 30.5)
df['IMC'] = df['IMC'].astype(str).str.replace(',', '.').astype(float)

# Eliminar sufijos no numéricos ("180mg" -> 180.0, "45 años" -> 45.0)
df['Glucosa'] = df['Glucosa'].astype(str).str.replace('mg', '', regex=False).astype(float)
df['Edad'] = df['Edad'].astype(str).str.replace(' años', '', regex=False).astype(float)

# Normalizar categorías (Quitar espacios extra y poner mayúscula inicial)
df['Actividad_Fisica'] = df['Actividad_Fisica'].str.strip().str.capitalize()
df['Fumador'] = df['Fumador'].str.strip().str.capitalize()

# ==========================================
# FASE 3: ESTRATEGIAS PARA VALORES NULOS (NaN)
# ==========================================

# ESTRATEGIA A: ELIMINACIÓN (Dropna)
# Descomenta esta línea si tu dataset es enorme y prefieres borrar el registro completo
# Si le falta la 'Edad' o la 'Glucosa', borra toda la fila.
# df.dropna(subset=['Edad', 'Glucosa'], inplace=True)

# ESTRATEGIA B: IMPUTACIÓN (Fillna) -> Usada aquí porque tenemos pocos datos
# Llenar edades vacías con la mediana de las edades (menos afectada por valores extremos)
df['Edad'].fillna(df['Edad'].median(), inplace=True)

# Llenar Glucosa vacía con la media aritmética
df['Glucosa'].fillna(df['Glucosa'].mean(), inplace=True)

# Llenar variable de texto vacía con el valor más repetido (la moda)
df['Fumador'].fillna(df['Fumador'].mode()[0], inplace=True)

# Verificar que ya no haya nulos
print(df.isnull().sum())

El dataset ahora está perfectamente estructurado y libre de ruido.
