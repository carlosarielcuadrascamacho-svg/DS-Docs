Ingeniería de Datos

Análisis, diagnóstico y saneamiento del dataset Salud.csv. Transformación de datos crudos (Raw Data) en un formato estructurado apto para algoritmos de Machine Learning.

1. Diagnóstico de Anomalías

Los modelos matemáticos (como la Regresión Lineal o los Árboles de Decisión) fallan catastróficamente si reciben texto donde esperan números, o si encuentran valores vacíos (NaN). Al explorar el archivo original, detectamos 4 problemas críticos que deben ser resueltos en el pipeline.

Errores Detectados en Salud.csv

  • Tipos Mixtos: 180mg en la columna Glucosa, 45 años en Edad.
  • Errores de Localización: Uso de coma europea "30,5" en lugar de punto decimal.
  • Inconsistencia Categórica: " baja ", "NO", "si" (espacios y mayúsculas aleatorias).
  • Valores Nulos (NaN): Filas sin edad (,28.9,112...) o sin glucosa (51.0,18.8,80,,Media,No).

2. Desarrollo del Pipeline (Saneamiento)

¿Qué estamos haciendo? Estamos pasando un "filtro" automatizado por todo el archivo Excel. Le decimos a Python: "Si ves la palabra 'mg', bórrala. Si ves una coma, cámbiala por un punto para que sea un número real. Y si falta la edad de alguien, ponle la edad promedio de los demás para no tener que borrar a ese paciente completo."

Conceptos Clave (Data Wrangling):

  • Casteo y Regex: Usamos .str.replace() para eliminar caracteres alfanuméricos de vectores numéricos, permitiendo la conversión posterior con .astype(float).
  • Normalización Categórica: .str.strip().str.capitalize() elimina espacios en blanco incidentales (leading/trailing whitespaces) y unifica el capitalizado para evitar que el modelo trate "si", "Si" y " SI" como tres clases distintas.
  • Imputación Robusta: En lugar de usar dropna() (que reduce la muestra), imputamos valores continuos usando la mediana (menos sensible a outliers) y valores categóricos usando la moda. 

import pandas as pd
import numpy as np

# 1. Cargar el dataset crudo
df = pd.read_csv('Documentos/Salud.csv')

# ==========================================
# FASE 1: LIMPIEZA ESTRUCTURAL (STRING A FLOAT)
# ==========================================
# Cambiar coma por punto en el IMC ("30,5" -> 30.5)
df['IMC'] = df['IMC'].astype(str).str.replace(',', '.').astype(float)

# Eliminar sufijos no numéricos ("180mg" -> 180.0, "45 años" -> 45.0)
df['Glucosa'] = df['Glucosa'].astype(str).str.replace('mg', '', regex=False).astype(float)
df['Edad'] = df['Edad'].astype(str).str.replace(' años', '', regex=False).astype(float)

# ==========================================
# FASE 2: NORMALIZACIÓN CATEGÓRICA
# ==========================================
# Quitar espacios extra y poner la primera letra en mayúscula (" baja " -> "Baja", "NO" -> "No")
df['Actividad_Fisica'] = df['Actividad_Fisica'].str.strip().str.capitalize()
df['Fumador'] = df['Fumador'].str.strip().str.capitalize()

# ==========================================
# FASE 3: IMPUTACIÓN DE VALORES NULOS (NaN)
# ==========================================
# Llenar edades vacías con la mediana de las edades
df['Edad'].fillna(df['Edad'].median(), inplace=True)

# Llenar Glucosa vacía con la media
df['Glucosa'].fillna(df['Glucosa'].mean(), inplace=True)

# Llenar Fumador vacío con el valor más común (la moda)
df['Fumador'].fillna(df['Fumador'].mode()[0], inplace=True)

# Verificar que ya no haya nulos
print(df.isnull().sum())
